## 6. RECOMENDACOES PARA PROXIMAS SPRINTS

### Sprint Imediata — P0 (SEM ISSO O PRODUTO NAO EXISTE)

**Implementar geracao REAL de imagem com IA no editor**
- Stack ja existe: Replicate (Flux) esta na arquitetura do CriativoAI
- Fluxo: usuario preenche produto + estilo + copy → IA gera imagem no canvas
- Diferencial: usar o copy gerado pelo Groq como prompt para o Flux
- Estimativa: 3-5 dias com Claude Code
- Fonte de referencia: como AdCreative.ai funciona na pratica

### Sprint 1 — P1 (Diferenciadores core)

**1. Creative Scoring (versao simplificada)**
- Nao precisa de CNN complexo no MVP
- MVP viavel: usar Groq/Claude para avaliar o criativo (elementos presentes, hierarquia visual, CTA, contraste)
- Score de 0-100 com feedback textual ("seu CTA esta pouco visivel", "adicione desconto junto ao CTA")
- Treinado com frameworks de CRO, nao em 450M ads
- Estimativa: 4-6 dias
- Valor percebido: ENORME — usuarios adoram saber qual criativo vai performar melhor

**2. Scan de site para copy** (quick win)
- Usuario coloca URL → Groq faz scraping via Firecrawl → gera copy automatica
- Firecrawl.dev ja foi mapeado como ferramenta util nas pesquisas
- Estimativa: 1-2 dias
- Stack: Firecrawl API + Groq

**3. Geracao em lote (2-4 variantes)**
- Nao precisa ser 20+ no MVP
- Gerar 2-4 variantes diferentes do mesmo criativo (diferentes paletas, layouts)
- Loop na API que ja existe
- Estimativa: 2-3 dias

**4. Product Photoshoot AI (versao basica)**
- Upload foto produto → Flux img2img → produto em cenario profissional
- Usar Replicate Flux (ja na stack)
- Presets de cenario: branco clean, cafe, natureza, lifestyle
- Estimativa: 3-4 dias
- ALTO valor para publico e-commerce e dropshipping

### Sprint 2 — P2 (Crescimento e retencao)

**5. Multi-brand por conta**
- Pequena mudanca no schema Supabase
- Usuarios Starter: 1 marca, Pro: 3 marcas
- Estimativa: 1-2 dias

**6. Import de marca via URL**
- Firecrawl scrapa o site → extrai cores, logo, tom de voz → pre-preenche Brand Kit
- Killer feature para reducao de fricao no onboarding
- Estimativa: 2-3 dias com Firecrawl + Claude

**7. Redimensionamento 1-clique**
- Criativo pronto → 1 clique → gera versoes em todos os 8 formatos
- Estimativa: 2-3 dias (canvas resize + reposicionamento de elementos)

**8. Ad fatigue notification**
- Email/notificacao apos 14 dias ("Seu criativo pode estar cansando — crie variantes")
- Loop de retencao simples
- Estimativa: 1 dia

**9. Competitor insights (versao basica)**
- Input: URL do concorrente
- Firecrawl + scraping Meta Ad Library (nao-oficial) OU Groq analisa site
- Entrega: ton of voice, cores predominantes, mensagens principais
- Estimativa: 3-5 dias (versao simplificada)

### Sprint 3 — P3 (Escala e premium)

**10. Integracao Meta Ads** (publicar direto)
- OAuth Facebook, Meta Marketing API
- Alta complexidade, mas elimina maior fricao do usuario
- Estimativa: 5-7 dias
- Prerequisito: produto validado com usuarios pagantes

**11. UGC-style video (avatar)**
- HeyGen API ou similar
- Usuario: foto produto + script → video avatar falando
- Preco HeyGen: $24-48/mes para integracao
- Estimativa: 3-4 dias de integracao

**12. Product Videoshoot**
- Runway ML ou Kling API
- Foto produto → video cinematografico 10s
- Estimativa: 3-4 dias de integracao

### VANTAGEM COMPETITIVA REAL DO CRIATIVOAI (MANTER E EXPLORAR)

1. **Preco 4-7x menor** — $10-20 vs $39-149 — killer para mercado BR
2. **Chat de suporte com IA** — AdCreative.ai NAO tem isso! Feature unica.
3. **Frameworks de copy mais ricos** — CriativoAI tem 5 frameworks vs 2 (AIDA+PAS) do AdCreative
4. **Sem billing predatorio** — maior reclamacao do AdCreative e cobrancas surpresa. CriativoAI pode fazer grande diferencial com billing transparente e generoso
5. **Foco no publico BR** — gestor de trafego, afiliado, dropshipper brasileiro — linguagem, suporte e templates especificos
6. **Onboarding mais simples** — AdCreative tem muitas opcoes que confundem; CriativoAI pode ser o "AdCreative para quem nao quer complicacao"
