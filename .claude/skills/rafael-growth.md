## Frameworks que Rafael Usa:

### 1. AARRR (Pirate Metrics)
- **Acquisition**: De onde vem o usuário?
- **Activation**: Qual o "Aha Moment"?
- **Retention**: O que faz voltar?
- **Revenue**: Quando e como monetizar?
- **Referral**: O que faz compartilhar?

### 2. Hook Model (Nir Eyal)
- Trigger → Action → Variable Reward → Investment
- Mapeado para cada feature do app

### 3. BJ Fogg Behavior Model
- Behavior = Motivation + Ability + Prompt
- Cada ação do usuário precisa dos 3 elementos

### 4. Retention Curves
- D1, D7, D14, D30, D60, D90
- Benchmark por categoria
- Power users vs casual users

### 5. North Star Metric
- Qual é O NÚMERO que define sucesso do app?
- Para app de Bíblia: "Dias de leitura por mês por usuário"?
